Aller Leer Tutorial Course




Browse Courses :

  1. Click on Browse Courses










  2. Select the Course You Would like to view










  3. Search for Specific Course










  4. If you like the course give it a Thumbs up !!!










  5. If you dont like the course or it has flaws Give it a Thumbs Down...










  6. Return to main menu and view the Course !!!










  7. When done , remove the course from you "Owned Courses" Panel










  8. Go and Learn More !!!










Create Course :

  1. Click on Create New Course Button










  2. Select The element you want to add










  3. Edit the element to suit your needs










  4. Save the course to edit it later










  5. Load the course from the "Saved Courses" folder










  6. Preview Course in the web browser !










  7. Publish the Course for all the See =)
